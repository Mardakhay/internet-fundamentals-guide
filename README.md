# Internet Fundamentals — Interactive Study Guide

A single-file interactive HTML study guide built from Notion notes covering how the internet works — from packets and protocols to browsers and web hosting.


## Getting Started

1. Download html file.
2. Open it in any modern browser (Chrome, Edge, Firefox, Safari)
3. No installation, no dependencies, no internet connection required

---

## Features

### Navigation & Layout
- Fixed sidebar with smooth-scroll navigation and an active-section highlighter as you scroll
- Animated top progress bar showing how far through the guide you've read
- Mobile responsive with a hamburger menu for small screens

### Interactive Elements
- **Core Terminology** — 10 hover-animated flashcard-style cards, one per key term
- **DNS 8-Step Lookup** — click any step to highlight it and trace the full resolution chain
- **DNS Caching Flow** — clickable nodes showing Browser → OS → ISP → Upstream progression
- **HTTP Evolution Timeline** — click any version (0.9 → 1.0 → 1.1 → 2) to expand its details, including the animated TCP three-way handshake visual for HTTP/1.0

### Rich Content Sections
- **Protocols** — color-coded protocol pills (IP, TCP, UDP, DNS, HTTP/S) with descriptions
- **IPv4 vs IPv6** — side-by-side comparison boxes with real address examples
- **DNS Server Types** — 4 numbered cards in a responsive grid
- **TLS 3-Step Process** — certificate → handshake → encryption cards
- **Port Reference Table** — HTTP, HTTPS, FTP, SMTP, SSH
- **Hosting Types Table** — Shared, VPS, Dedicated, Cloud with performance badges
- **Browser Request Pipeline** — 6-step numbered flow from DNS lookup to JavaScript execution
- **Browser Components Diagram** — 5 interactive cards (UI, Rendering Engine, Networking, JS Engine, Security)
